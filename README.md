# JavaFX Project

This is a JavaFX application project for Microprocessors Team 26.

## Prerequisites

- Java 8 or higher
- Maven 3.6 or higher

## Building and Running

The project includes Maven Wrapper, so Maven will be downloaded automatically if not installed.

1. Compile the project:
   ```
   ./mvnw compile
   ```
   (On Windows: `mvnw.cmd compile`)

2. Run the application:
   ```
   ./mvnw exec:java
   ```
   (On Windows: `mvnw.cmd exec:java`)

## Project Structure

- `src/main/java`: Source code
- `src/main/resources`: Resources
- `src/test/java`: Test code

The main class is `com.example.App`.
